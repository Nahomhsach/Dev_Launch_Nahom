About Me

Hi there! I'm Nahom and I am a highly motivated high school student with a passion for technology and coding. Over the past three years, I have taken several computer science courses that have shifted my view of software engineering radically. I will not forget the first time that I saw my code execute a program in Java, or learned the tenets of object-oriented programming and how they propelled my ability to architect solutions, or made my very first Webpage and suddenly felt like an artist returning to interactive Web development with HTML, CSS, and JavaScript! Each experience was a building block for my technical training and knowledge, yet I came away with much broader lessons on problem-solving, perseverance and the habit of lifelong learning. I learned that programming is not simply about writing codes, it is about finding a solution that works for someone. It's about being part of something larger and spanning meaningful solutions to real problems that affect people's lives. As I continue on my software engineeering path I look forward to learning more advanced technologies and concepts, and exploring opportunities to participate in projects that benefit our society.

My Programming Journey

9th Grade - Java (Mrs. Pathan)

My first experience with programming was truly eye-opening. I learned the fundamental building blocks like loops, variables, and conditionals in Java, which helped me understand how computers process instructions. I still remember the excitement of writing my first 'Hello World' program and seeing it run successfully! Through hands-on practice, I created programs that could perform calculations, process user input, and even built a simple text-based game that taught me how to think logically and solve problems step by step

10th Grade - Object Oriented Programming (Mr. Dal)

This class helped me understand how to build organized programs using concepts like classes, inheritance, and encapsulation. Learning these principles transformed how I approach software development - instead of writing monolithic code, I learned to break down complex problems into manageable, reusable components. I applied these concepts by creating a student management system where I used inheritance to handle different types of users and encapsulation to protect sensitive data. This project showed me how proper organization makes code more maintainable and scalable.

11th Grade - Web Design (Mrs. Pathan)

Currently learning HTML, CSS, and JavaScript, and I'm amazed by how these technologies work together to create engaging user experiences. I'm building websites that are responsive and interactive! My projects include a personal portfolio that adapts seamlessly to different screen sizes and a dynamic to-do list application that uses local storage to persist data. I've also experimented with animations and transitions to make my websites more engaging. The immediate visual feedback of web development is incredibly satisfying, and I love how it combines creativity with technical skills.
